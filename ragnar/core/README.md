# Core Modules

Core modules to implement first:

- `role_registry.py`: load and validate role contracts.
- `approval_broker.py`: enforce read/free and outward/approval-required actions.
- `audit_log.py`: append-only audit trail.
- `handoff.py`: typed role-to-role communication.
- `run_state.py`: durable run/task/checkpoint model.
- `worktree_manager.py`: isolated branch and worktree allocation.
